## חלק 4: יצירת Cron Job - המערכת הפועלת

### שקף 9: מהו Cron Job?
**Cron = מתזמן משימות באוטומטי**

```
┌─────────────────────────────────────┐
│  כל 5 שניות:                       │
│  1. קרא משימות מ-Todoist           │
│  2. שלח ל-Cursor AI                │
│  3. Cursor מבצע את המשימה           │
│  4. עדכן תוצאות ב-Todoist          │
└─────────────────────────────────────┘
```

**למה כל 5 שניות?**
- ⚡ תגובה מהירה למשימות חדשות
- 🔄 עדכונים בזמן אמת
- 📊 מעקב צמוד אחר התקדמות

---

### שקף 10: תרשים זרימה (Flow Diagram)
```
┌──────────────┐
│  Todoist     │
│  (משימות)    │
└──────┬───────┘
       │ API Call

       ▼
┌──────────────┐

       │ Send Tasks

       │ Results
       │
       ▼
┌──────────────┐
│  Update      │
│  Todoist     │
└──────────────┘
```

### שקף 11: דוגמת קוד - Cron Script
```python
import schedule
import time
from todoist_api import TodoistAPI
from cursor_agent import CursorAgent

# Initialize
todoist = TodoistAPI(token="YOUR_TOKEN")
cursor = CursorAgent()

def process_tasks():
    """מעבד משימות כל 5 שניות"""
    # 1. קרא משימות
    tasks = todoist.get_tasks()
    
    # 2. לכל משימה
    for task in tasks:
        if not task.is_completed:
            # 3. שלח ל-Cursor
            result = cursor.execute(task.content)
            
            # 4. עדכן ב-Todoist
            task.add_comment(f"תוצאה: {result}")
            task.complete()
    
    print(f"✓ עובדו {len(tasks)} משימות")

# הרץ כל 5 שניות
schedule.every(5).seconds.do(process_tasks)

while True:
    schedule.run_pending()
    time.sleep(1)
```
